Enumerable

The Hits class is an enumeration for Hit objects. The Hits object never actually creates all of the Hit structures at once. Instead it lazily creates an object when stepping through the each method or the array based lookup. The structure is built for a specific term, the start index and end index in the suffix array, the fulltext reader and the suffix array reader used to lookup the offsets.

def [](index)
  index += self.size if index < 0
  index = from_index + index
  if index < to_index && index >= from_index
    info = suffix_array_reader.index_to_info(index)
    Hit.new(self.term, index, info[0], info[1], info[2], self.fulltext_reader)
  else
    nil
  end
end

def each
  term = self.term
  fulltext = self.fulltext_reader
  suffix_array = self.suffix_array_reader
  self.from_index.upto(self.to_index - 1) do |index|
    info = suffix_array_reader.index_to_info(index)
    yield Hit.new(term, index, info[0], info[1], info[2], fulltext)
  end
end
